Sloop frequently-asked questions

What's the point of Sloop?

Sloop is a free Android browser for Open Data. A normal web browser is designed for documents full of text and pictures; Sloop is designed to make it easy (well, as easy as possible) to browse structured, fielded data on the web.

Structured data isn't as pretty as documents, but you can do things with it like filtering, sorting, summing up, etc. To read more reasons why open data is a good thing, visit some like the International Aid Transparency Initiative, the Open Government Partnership, data.gov (US), data.gov.uk (Britain), or data.gc.ca (Canada).

What formats does Sloop support?
Sloop reads data collections in a simple format called Comma-Separated Values (CSV). You can save CSV from any spreadsheet or database program, and you can also create CSV data collections using Google Docs. For more information, see Publishing data.
How do I link to a different dataset?
Sloop assumes that any field with a value that looks like a URL (with no extra text) actually is one. When the user taps the link, Sloop will check to see if the resource on the other end has a content type of "text/csv"; if so, it will try to load it as a data collection; otherwise, it will send it off to the Android browser (or some other app that can handle it).
How can I publish CSV straight from a Google spreadsheet?
Select "Share" for your spreadsheet and set its status to "Public on the web." Then copy the sharing link and add "output=csv&" immediately after the question mark. For example, if the sharing link is "https://docs.google.com/spreadsheet/ccc?key=0AoDV0i2WefMXdEI2VV9Xb1I5eFpBeS1HYkw5NGNqR3c&usp=sharing", you can get a CSV link by using https://docs.google.com/spreadsheet/ccc?output=csv&key=0AoDV0i2WefMXdEI2VV9Xb1I5eFpBeS1HYkw5NGNqR3c&usp=sharing.
What languages does Sloop support?
The Sloop user interface is available in English, French, Spanish, and German. I'd love to add more languages — if you can help translate, let me know, and I'll send you a simple text file of strings (no programming knowledge required). I have not yet tested Sloop in a right-to-left language like Arabic, Hebrew, or Mandarin, and I'd be excited to see how it works.
Why Android and not iOS?

Android is the world's most-popular mobile operating system, it's open source, and it's available on less-expensive devices, so (I hope) it will be more accessible in the developing world. I've been a Linux user for 20 years and a Java programmer for 16, so Android (which is based on Linux) was a comfortable fit.

That said, I'd be thrilled to see a Public-Domain Sloop port to other mobile operating systems, including iOS (for the iPhone and iPad), Blackberry OS, Windows RT (for the Microsoft Surface), and others, including desktop operating systems. If you're interested in working on any of these, please let me know, and I'll provide as much (non-monetary) support as I can.

Do you support visually-impaired users?
I don't know, but I'd like to. If you're interested in helping me make Sloop more accessible, through testing and feedback, please get in touch.
Why Public Domain?
If you think about it, any software license, even an Open-Source license, is actually a polite threat: "if you don't follow my rules, I'll sue you." I don't want to sue you, so I'm not making any rules that you can break. As a Public Domain app, Sloop is free — really free — so enjoy.
How can I help?

If you have data that might be useful to someone else, put it online. Lots of people (not just Sloop users) will be able to benefit. It's easy to get started — if you can use a spreadsheet program, you can publish open data. If you have nowhere to host your open data on the web, let me know, and I'll add it to the Sloop site here.

If you speak a language we don't support, volunteer to help translate (no programming knowledge required).

If you're a software developer, check us out on github. If you haven't tried Android coding before, this is a good place to start (I write lots of comments); if you're an experienced Android developer, then you can teach me, since this is my first Android project. Make your own branch, then let me know when you have something to contribute back to the main app. I'll give all contributors (programmers or otherwise) credit on a screen in the app.

No matter what skills you have, you can help by using Sloop, sending feedback (such as bug reports or feature suggestions), and getting your friends and colleagues to try it out.

Why the name "Sloop"?
Sloop is named after not the modern sailing sloop, but the Sloop-of-war from two centuries ago. The Sloop was a fast ship-rigged (three-masted) vessel, smaller than a frigate or a ship of the line, but still capable of crossing the ocean. The British Royal Navy often used sloops to carry dispatches quickly from point to point all over the world — it seemed like a nice analogy for a small, light data browser.
