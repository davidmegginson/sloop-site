Sloop data browser

Sloop is a Public-Domain Android mobile browser for open data. It presents spreadsheet-type data in a mobile-friendly way:

  • Swipe left or right to move through records.
  • Swipe up or down to scroll through the fields in a single record.
  • Long press on a single field to make it into a filter (so that you'll see only records that have the same value in the same field, for example, records where the Country field has the value "UK").
  • Follow links to other data collections, or to web sites.
